## 📋 方案概览

### 核心目标

1. **职责清晰**：前端只负责 UI 交互，后端负责业务逻辑
2. **能力统一**：所有用户输入都由后端 LLM 统一解析
3. **体验一致**：用户无需记忆特定格式，自然语言即可
4. **易于维护**：单一解析逻辑，降低维护成本

### 预期收益

- 代码量减少 40%
- 解析准确率提升 58%（从 60% 到 95%）
- 维护成本降低 50%
- 用户体验显著提升
